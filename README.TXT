# 🏰 Park of Zuul

A Java text adventure game built to demonstrate object-oriented programming concepts.

---

## 📖 Project Overview

This project demonstrates:

- ✅ Object-Oriented Programming
- ✅ Inheritance
- ✅ Abstract Classes
- ✅ Encapsulation
- ✅ Polymorphism

---

## 🚀 Features

- Explore multiple rooms
- Pick up items
- Interact with objects
- Command parser
- Inventory system

---

## 🛠 Technologies

- Java
- IntelliJ IDEA
- Git
- GitHub
