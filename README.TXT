# 🏰 Park of Zuul

A Java text adventure game built to demonstrate object-oriented programming concepts.

---

## 📖 Project Overview

This project demonstrates:

- ✅ Object-Oriented Programming
- ✅ Inheritance
- ✅ Abstract Classes
- ✅ Encapsulation
- ✅ Polymorphism

---

## 🚀 Features

- Explore multiple rooms
- Pick up items
- Interact with objects
- Command parser
- Inventory system

---

## 🛠 Technologies

- Java
- IntelliJ IDEA
- Git
- GitHub

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/ParkOfZuul.git


2. Open the project in a Java enviroment. Eg: VSCode, Netbeans, IntelliJ, etc.

3. Build the project.

4. Run the `Game.java` (or main class).
